A simple full-stack dashboard application built with Next.js.

This project follows the App Router approach and is based on the official Next.js dashboard learning project. It demonstrates how to build a modern web application with routing, layouts, authentication, database integration, search, pagination, and server-side data updates.

## Features

- Public home page
- Login page
- Protected dashboard routes
- Invoice management
  - Create invoices
  - Edit invoices
  - Delete invoices
- Search and pagination
- Server-side data fetching
- Server Actions for mutations
- Database integration
- Responsive dashboard layout

## Tech Stack

- Next.js
- React
- TypeScript
- App Router
- PostgreSQL
- CSS / component-based UI

## What I Learned

This project helped me understand:

- File-based routing with the Next.js App Router
- Nested layouts and shared dashboard structure
- Server Components and server-side data fetching
- Streaming and loading states
- Search and pagination with URL parameters
- Updating data with Server Actions
- Authentication and route protection
- Organizing a real-world Next.js project structure
